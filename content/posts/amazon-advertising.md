# 亚马逊广告投放 - 详细链路

```
广告架构搭建 → 投放执行 → 数据监控 → 优化调整 → 规模扩张
      ↑________________________________________________↓
```

---

## 一、广告架构搭建（基础设置）

### 1.1 输入
- 产品Listing
- 关键词库
- 竞品ASIN
- 预算规划

### 1.2 处理流程

```
Step 1：关键词分层
├── 品牌词（Brand Terms）
│   ├── 品牌名
│   ├── 品牌+产品
│   └── 品牌+型号
├── 核心词（Core Keywords）
│   ├── 产品核心词（高搜索量）
│   ├── 产品长尾词（精准转化）
│   └── 场景词（使用场景）
├── 竞品词（Competitor Terms）
│   ├── 竞品品牌名
│   └── 竞品产品名
└── 否定词（Negative Keywords）
    ├── 不相关词
    ├── 低转化词
    └── 高花费无订单词

Step 2：广告活动规划
├── 自动广告（4个活动）
│   ├── 紧密匹配 - 高竞价
│   ├── 宽泛匹配 - 中竞价
│   ├── 同类商品 - 中竞价
│   └── 关联商品 - 低竞价
├── 手动-广泛（1-3个活动）
│   └── 核心词+修饰词
├── 手动-词组（3-5个活动）
│   └── 高潜力词
├── 手动-精准（5-10个活动）
│   └── 高转化词
└── SD广告（2个活动）
    ├── 商品投放
    └── 受众投放

Step 3：竞价策略设置
├── 动态竞价-仅降低
│   └── 用于：测试期、控制成本
├── 动态竞价-提高和降低
│   └── 用于：成熟期、争取曝光
├── 固定竞价
│   └── 用于：精准控制、防守词
└── 竞价调整
    ├── 首页顶部溢价（+50-100%）
    ├── 商品页面溢价（+20-50%）
    └── 根据表现动态调整

Step 4：预算分配
├── 新品期（0-30天）
│   ├── 自动广告：40%
│   ├── 手动广泛：30%
│   ├── 手动精准：20%
│   └── SD广告：10%
├── 成长期（30-90天）
│   ├── 自动广告：20%
│   ├── 手动广泛：30%
│   ├── 手动精准：40%
│   └── SD广告：10%
└── 成熟期（90天+）
    ├── 自动广告：10%
    ├── 手动广泛：20%
    ├── 手动精准：50%
    └── SD广告：20%
```

### 1.3 输出
- 广告活动结构表
- 关键词分层表
- 预算分配表

### 1.4 工具
| 工具 | 功能 |
|------|------|
| Sif | 关键词反查 |
| 卖家精灵 | 竞品广告分析 |
| 亚马逊广告后台 | 广告搭建 |

## 二、投放执行（日常操作）

### 2.1 输入
- 搭建好的广告活动
- 每日预算

### 2.2 处理流程

```
每日操作（15-30分钟）
├── 检查预算消耗
│   ├── 是否提前耗尽（加预算或降竞价）
│   └── 是否消耗不足（加竞价或拓词）
├── 检查ACOS
│   ├── 异常飙升（检查是否有异常点击）
│   └── 持续高位（优化或暂停）
├── 检查曝光/点击
│   ├── 曝光骤降（竞价不足或词问题）
│   └── 点击率低（主图或价格问题）
└── 检查转化
    ├── 转化率高（考虑加预算）
    └── 转化率低（Listing问题）

每周操作（1-2小时）
├── 搜索词报告分析
│   ├── 下载报告（过去7天）
│   ├── 筛选高ACOS词（>50%）
│   ├── 筛选高转化词（有订单）
│   └── 筛选高花费无订单词
├── 否定词优化
│   ├── 不相关词→精准否定
│   └── 低转化词→词组否定
├── 新词挖掘
│   ├── 自动广告出单词→手动投放
│   └── 竞品词→测试投放
└── 竞价调整
    ├── 高ACOS词→降竞价（10-20%）
    └── 低ACOS词→加竞价（10-20%）

每月操作（半天）
├── 广告结构复盘
│   ├── 活动表现对比
│   ├── 关键词表现分析
│   └── 广告位效果分析
├── 预算重新分配
│   ├── 砍掉表现差的活动
│   └── 加大表现好的活动
├── 竞品监控
│   ├── 竞品广告变化
│   └── 竞品排名变化
└── 下月计划
    ├── 预算规划
    └── 优化重点
```

### 2.3 输出
- 每日检查记录
- 每周优化报告
- 每月复盘总结

## 三、数据监控（效果追踪）

### 3.1 核心指标

| 指标 | 公式 | 健康值 | 说明 |
|------|------|--------|------|
| **曝光量** | Impressions | - | 流量基础 |
| **点击量** | Clicks | - | 流量质量 |
| **点击率** | CTR = Clicks/Impressions | >0.3% | 主图吸引力 |
| **花费** | Spend | - | 广告成本 |
| **销售额** | Sales | - | 广告产出 |
| **ACOS** | ACOS = Spend/Sales | <25% | 广告效率 |
| **ROAS** | ROAS = Sales/Spend | >4 | 投资回报 |
| **转化率** | CVR = Orders/Clicks | >10% | Listing质量 |
| **CPC** | CPC = Spend/Clicks | 因品类而异 | 单次点击成本 |
| **CPA** | CPA = Spend/Orders | - | 单次获客成本 |

### 3.2 监控维度

```
按时间维度
├── 日数据：异常监控
├── 周数据：趋势分析
├── 月数据：效果评估
└── 季度数据：战略调整

按广告类型
├── SP自动：测词效果
├── SP手动：转化效果
├── SD广告：再营销效果
└── SB广告：品牌效果

按关键词
├── 品牌词：防守效果
├── 核心词：流量效果
├── 长尾词：转化效果
└── 竞品词：截流效果

按广告位
├── 首页顶部：曝光价值
├── 商品页面：转化价值
└── 搜索结果：综合价值
```

### 3.3 数据报表模板

| 日期 | 活动 | 曝光 | 点击 | CTR | 花费 | 销售 | ACOS | 订单 | CVR |
|------|------|------|------|-----|------|------|------|------|-----|
| 示例 | 自动-紧密 | 10000 | 50 | 0.5% | $30 | $120 | 25% | 5 | 10% |

## 四、优化调整（提升效率）

### 4.1 优化方向

```
方向1：降低ACOS
├── 否定无效词
├── 降低高ACOS词竞价
├── 优化Listing提升转化
├── 调整广告位溢价
└── 分时调价（高转化时段加价）

方向2：提升销售额
├── 增加预算
├── 提高竞价获取更多曝光
├── 拓展新关键词
├── 开启SD广告
└── 优化主图提升CTR

方向3：提升利润率
├── 精准投放（减少浪费）
├── 长尾词策略（低成本高转化）
├── ASIN定位（精准流量）
├── 品牌词防守（低成本）
└── 动态竞价优化
```

### 4.2 具体优化动作

| 问题 | 诊断 | 解决方案 |
|------|------|----------|
| 曝光低 | 竞价太低 | 提高竞价20-50% |
| 点击率低 | 主图不吸引 | 优化主图、价格调整 |
| ACOS高 | 词不准或转化差 | 否定词、优化Listing |
| 无转化 | Listing问题 | 优化文案、图片、价格 |
| 预算耗尽快 | 竞价太高 | 降低竞价或分时投放 |
| 预算花不完 | 竞价太低或词少 | 提高竞价、拓词 |

### 4.3 优化节奏

```
新品期（0-30天）
├── 目标：跑词、出单
├── 策略：
│   ├── 自动广告全开
│   ├── 不急于否定（收集数据）
│   ├── 关注出单词
│   └── 容忍高ACOS（<50%）
└── 优化重点：
    ├── 每天检查预算
    └── 每周下载搜索词报告

成长期（30-90天）
├── 目标：优化ACOS、提升销量
├── 策略：
│   ├── 否定无效词
│   ├── 自动出单词转手动
│   ├── 测试不同竞价
│   └── 开启SD广告
└── 优化重点：
    ├── 每周否定词优化
    ├── 每两周竞价调整
    └── 每月结构复盘

成熟期（90天+）
├── 目标：利润最大化
├── 策略：
│   ├── 精准投放
│   ├── 品牌广告
│   ├── 防守策略
│   └── 再营销
└── 优化重点：
    ├── 精细化调整
    ├── 季节性优化
    └── 竞品监控
```

## 五、规模扩张（放大效果）

### 5.1 扩张条件

- ACOS < 利润率（广告盈利）
- 库存充足（不断货）
- Review > 20个（有基础）
- 排名稳定（前50）

### 5.2 扩张策略

```
策略1：预算扩张
├── 逐步增加预算（每周+20%）
├── 监控ACOS变化
└── 找到最佳预算点

策略2：关键词扩张
├── 拓展长尾词
├── 测试新词
└── 竞品词投放

策略3：广告类型扩张
├── 开启SD广告
├── 开启SB广告
└── 测试SBV视频广告

策略4：市场扩张
├── 同产品多站点
├── 变体扩展
└── 关联产品投放
```

## 七、关键决策点

| 决策点 | 决策内容 | 决策依据 |
|--------|----------|----------|
| 预算设置 | 每日预算多少 | 利润率、库存、资金 |
| 竞价设置 | 初始竞价多少 | 建议竞价、利润空间 |
| 否定词 | 哪些词要否定 | 搜索词报告、相关性 |
| 暂停/开启 | 活动是否暂停 | ACOS、ROI、库存 |
| 扩张时机 | 何时加预算 | ACOS稳定、库存充足 |
| 收缩时机 | 何时降预算 | ACOS飙升、库存紧张 |

## 九、常见问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 广告花不出去 | 竞价太低、词太窄 | 提高竞价、拓词 |
| ACOS过高 | 词不准、转化差 | 否定词、优化Listing |
| 有点击无转化 | Listing问题 | 优化图片、价格、文案 |
| 曝光突然下降 | 竞价被超、库存不足 | 检查竞价、库存 |
| 竞品恶意点击 | 竞争对手 | 开Case、调整策略 |
| 广告费占比过高 | 依赖广告 | 优化自然排名、站外引流 |
