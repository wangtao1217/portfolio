# FBA发货与物流 - 详细链路

> **面试考点**: 「如何发货到FBA？」「头程物流怎么选？」「发货计划怎么做？」
> **实操场景**: 入职第一天可能就要操作发货，必须熟练掌握

---

## 一、发货前准备

### 1.1 创建发货计划前置检查

| 检查项 | 操作路径 | 合格标准 |
|--------|----------|----------|
| 商品可售状态 | 库存→管理FBA库存 | 状态为「可售」或「需补货」 |
| Listing完整度 | 库存→编辑 | 标题、图片、五点、A+完整 |
| 危险品审核 | 库存→管理危险品分类 | 非危险品或已通过审核 |
| 类目审核 | 库存→编辑→重要信息 | 无类目限制提示 |
| 品牌白名单 | 品牌注册→品牌优势 | 品牌已备案或申请白名单 |

### 1.2 包装与标签准备

**包装要求**:
- 外箱：六面硬质纸箱，禁止使用带钉纸箱
- 尺寸：单边不超过63.5cm，重量不超过22.5kg（超重需贴「Team Lift」标签）
- 填充：使用充气袋、泡沫等填充物，防止商品晃动

**标签要求**:
```
必需标签：
├─ FNSKU标签（商品标签）→ 贴在每个商品包装上
├─ 外箱标签（Shipment ID）→ 贴在外箱侧面，每箱2张
├─ Made in China标签 → 外箱和商品包装都需标注
└─ 超重标签（如适用）→ 超过15kg需贴「Team Lift」，超过22.5kg需贴「Mech Lift」
```

**标签打印规格**:
- FNSKU：50×25mm或50×30mm热敏纸
- 外箱标签：A4纸打印，尺寸100×100mm
- 打印机：热敏打印机（推荐Zebra GK888t）或激光打印机

## 二、创建发货计划

### 2.1 后台操作流程

```mermaid
flowchart LR
    A[库存→管理FBA库存] --> B[选择商品→发/补货]
    B --> C[创建新的入库计划]
    C --> D[选择包装类型]
    D --> E[填写发货数量]
    E --> F[确认入库计划]
    F --> G[打印标签]
    G --> H[确认发货]
```

**详细步骤**:

**Step 1**: 进入「库存」→「管理FBA库存」→ 勾选要发货的商品 → 点击「发/补货」

**Step 2**: 选择包装类型
- **混装商品**：同一箱内有不同SKU（选择「需要贴标」）
- **原厂包装**：同一箱内只有相同SKU（选择「无需贴标」）

**Step 3**: 填写发货数量
```
计算公式：
发货数量 = 目标库存天数 × 日均销量 - 现有库存 - 在途库存

示例：
- 目标库存天数：30天
- 日均销量：20件
- 现有库存：200件
- 在途库存：150件
- 发货数量 = 30×20 - 200 - 150 = 250件
```

**Step 4**: 确认入库配置
- **亚马逊优化货件拆分**：分仓，免入库配置费
- **部分货件拆分**：2-3个货件，低入库配置费
- **最低货件拆分**：1个货件，标准入库配置费

**Step 5**: 打印标签
- 商品标签（FNSKU）：贴在每个商品上
- 外箱标签：贴在外箱侧面，每箱2张

### 2.2 发货计划关键参数

| 参数 | 说明 | 面试考点 |
|------|------|----------|
| 入库配置费 | 2024年3月起收取，按件计费 | 「为什么分仓更便宜？」 |
| 低库存费 | 库存低于28天销量时收取 | 「如何避免低库存费？」 |
| 货件追踪号 | 物流单号，必须在发货后24小时内填写 | 「追踪号填错怎么办？」 |
| 预计送达时间 | 货件到达FC的预计日期 | 「迟到会有什么影响？」 |

## 三、头程物流选择

### 3.1 物流方式对比

| 方式 | 时效 | 价格 | 适用场景 | 风险等级 |
|------|------|------|----------|----------|
 **快递**（DHL/FedEx/UPS） | 3-7天 | 最高（30-40元/kg） | 紧急补货、高价值商品 | 低 |
 **空运** | 7-15天 | 高（25-35元/kg） | 季节性补货、中等货值 | 中 |
 **海运**（美森/普船） | 25-45天 | 低（8-15元/kg） | 大批量备货、低货值 | 中 |
 **铁路**（中欧班列） | 18-25天 | 中（12-18元/kg） | 欧洲线路、中等货量 | 中 |
 **卡航** | 15-20天 | 中高（15-22元/kg） | 欧洲线路、时效要求高 | 中 |

### 3.2 物流商选择标准

**考察维度**:
1. **时效稳定性**：承诺时效 vs 实际时效差异
2. **清关能力**：是否有自己的清关行，查验率如何
3. **赔付条款**：丢件/损坏的赔偿标准
4. **系统对接**：是否提供物流追踪API
5. **账期政策**：是否支持月结，账期多长

**推荐物流商**（仅供参考）:
- 快递：DHL（稳定）、FedEx（性价比高）
- 空运：云途、燕文、递四方
- 海运：美森（快船）、以星、长荣
- 欧洲：中欧班列（西安/重庆/成都出发）

### 3.3 物流成本计算

```
头程物流成本 = 运费 + 报关费 + 清关费 + 关税 + 目的港杂费

示例（海运1000kg到美国）：
- 运费：10元/kg × 1000kg = 10,000元
- 报关费：350元/票
- 清关费：500元/票
- 关税：（货值+运费）×税率（假设15%）
- 目的港杂费：约800元

单件成本 = 总成本 ÷ 发货数量
```

## 四、发货操作流程

### 4.1 贴标操作规范

**商品标签（FNSKU）贴法**:
- 位置：商品包装平整处，避免边角
- 覆盖：必须覆盖原有条形码（如有）
- 保护：使用透明胶带覆盖，防止磨损
- 数量：每个可售单位都必须贴标

**外箱标签贴法**:
- 位置：外箱侧面，不要贴顶面或底面
- 数量：每箱至少2张，对角线贴法
- 保护：用透明胶带完全覆盖，防水防磨损

### 4.2 称重与测量

```
必填信息：
├─ 单箱重量（实际称重）
├─ 单箱尺寸（长×宽×高，单位cm）
├─ 总箱数
└─ 每箱商品数量

注意事项：
- 重量精确到0.5kg
- 尺寸精确到1cm
- 数据必须准确，否则可能被拒收或收取额外费用
```

### 4.3 物流单号回填

**操作路径**：库存→货件→找到对应货件→「追踪货件」→填写物流信息

**必填字段**：
- 物流商名称
- 追踪编码（物流单号）
- 发货日期
- 预计送达日期

**时间要求**：
- 国内发货后24小时内必须填写
- 国际发货后48小时内必须填写
- 追踪号必须在货件送达前保持可查询状态

## 五、货件追踪与异常处理

### 5.1 货件状态监控

| 状态 | 含义 | 操作建议 |
|------|------|----------|
 **已创建** | 发货计划已创建，未发货 | 准备商品，安排物流 |
 **已发货** | 已填写追踪号，物流已揽收 | 监控物流轨迹 |
 **在途** | 货件运输中 | 关注清关状态 |
 **已送达** | 货件到达亚马逊FC | 等待接收确认 |
 **正在接收** | FC正在清点入库 | 通常3-7个工作日 |
 **已完成** | 商品已上架可售 | 检查库存数量是否准确 |
 **已关闭** | 货件处理完毕 | 归档记录 |

### 5.2 常见异常处理

**异常1：货件差异（数量不符）**
```
原因：
- 运输途中丢失
- 亚马逊清点错误
- 包装破损导致商品丢失

处理：
1. 查看货件差异报告
2. 联系物流商确认运输情况
3. 提交货件差异查询（货件→货件差异查询）
4. 如确认丢失，按物流商赔付条款索赔
```

**异常2：货件被拒收**
```
常见原因：
- 包装不符合要求（带钉纸箱、破损）
- 标签无法扫描（模糊、褶皱）
- 危险品未申报
- 超重/超尺寸未标注

处理：
1. 查看拒收原因（货件→问题日志）
2. 联系物流商退回或销毁
3. 重新包装后再次发货
4. 严重违规可能影响账号健康
```

**异常3：追踪号失效**
```
处理：
1. 联系物流商确认最新追踪号
2. 在货件详情页更新追踪信息
3. 如已无法追踪，提供物流商出具的运输证明
4. 避免因追踪号问题导致货件关闭
```

## 六、费用优化策略

### 6.1 降低入库配置费

| 策略 | 操作方式 | 节省效果 |
|------|----------|----------|
 **分仓发货** | 选择「亚马逊优化货件拆分」 | 免入库配置费 |
 **批量发货** | 单次发货数量≥100件 | 降低单件费用 |
 **避开高峰** | 避开Prime Day、黑五前2个月 | 降低仓储费 |
 **轻小商品** | 申请轻小商品计划 | 降低配送费 |

### 6.2 降低头程成本

```
策略1：海运+快递组合
- 大批量走海运（70%库存）
- 紧急补货走快递（30%库存）
- 平衡成本与库存风险

策略2：物流商谈判
- 月发货量≥10立方，可申请协议价
- 对比3家以上物流商报价
- 签订长期合作协议锁定价格

策略3：合理备货
- 根据销售预测精准备货
- 避免过量发货导致长期仓储费
- 滞销品及时清货或移除
```

## 七、面试高频问题

### Q1: 「如何创建FBA发货计划？」
**回答框架**:
1. 前置检查：确认商品可售、Listing完整、非危险品
2. 后台操作：库存→管理FBA库存→发/补货→创建计划
3. 填写信息：选择包装类型、填写数量、确认入库配置
4. 打印标签：商品标签（FNSKU）+ 外箱标签
5. 确认发货：选择物流方式、填写追踪号

### Q2: 「头程物流怎么选？」
**回答框架**:
1. 根据时效要求：紧急用快递，常规用空运，大批量用海运
2. 根据成本预算：高货值选快递，低货值选海运
3. 根据物流商评估：时效稳定性、清关能力、赔付条款
4. 示例：「我通常会根据库存天数和销售速度选择，比如库存低于14天时会选择空运紧急补货，常规备货选择美森快船」

### Q3: 「如何避免低库存费？」
**回答框架**:
1. 定义：库存低于28天销量时收取低库存费
2. 监控：每周查看库存绩效仪表板
3. 补货策略：保持库存可售天数在30-60天
4. 工具：使用亚马逊补货建议或第三方ERP

### Q4: 「货件差异怎么处理？」
**回答框架**:
1. 查看差异报告：确认差异数量和原因
2. 联系物流商：确认运输途中是否丢失
3. 提交查询：在后台提交货件差异查询
4. 索赔处理：根据物流商赔付条款申请赔偿
5. 预防措施：加强包装、购买物流保险

## 八、实操检查清单

### 发货前检查
- [ ] 商品Listing完整，图片、标题、五点无误
- [ ] FNSKU标签已打印，每箱2张外箱标签
- [ ] 外箱坚固，无破损，无钉子
- [ ] Made in China标签已贴
- [ ] 称重、测量数据准确记录

### 发货中检查
- [ ] 物流单号已获取
- [ ] 24小时内在后台填写追踪信息
- [ ] 保留物流底单和发票
- [ ] 监控物流轨迹，关注清关状态

### 发货后检查
- [ ] 货件状态变为「已送达」后跟进
- [ ] 确认商品上架数量和实际发货一致
- [ ] 如有差异，及时提交查询
- [ ] 归档发货记录，计算实际物流成本
